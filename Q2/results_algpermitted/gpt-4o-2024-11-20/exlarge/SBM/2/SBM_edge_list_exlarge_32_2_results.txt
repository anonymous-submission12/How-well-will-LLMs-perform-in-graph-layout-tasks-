Here is the Python code that leverages the `networkx` library to compute graph layouts. For maintaining consistency between Euclidean distance and graph-theoretic distance, the `spring_layout` algorithm is suitable because it attempts to represent distances between nodes in two-dimensional space proportional to their graph-theoretic distances.

The layout is then outputted in JSON format.

```python
import networkx as nx
import json

def generate_graph_layout(edge_list):
    # Create a graph from the edge list
    G = nx.Graph()
    G.add_edges_from(edge_list)

    # Generate the layout using spring algorithm
    # Spring layout tries to keep Euclidean distances proportional to graph-theoretic distances
    layout = nx.spring_layout(G)

    # Convert the layout to JSON-compatible format (dictionary of positions)
    layout_json = {str(node): pos.tolist() for node, pos in layout.items()}

    # Return layout in JSON format as a string
    return json.dumps(layout_json, indent=4)
```
